# Быстрый старт - Создание пиров WireGuard

## Шаг 1: Проверка и установка TUI

```bash
# Проверка наличия dialog
command -v dialog

# Если не найден, установите:
apt-get update
apt-get install -y dialog
```

Подробнее см. `TUI_SETUP.md`

## Шаг 2: Подготовка скрипта

```bash
# Сделать скрипт исполняемым
chmod +x wg-add-peer.sh

# Или скопировать в системную директорию
cp wg-add-peer.sh /usr/local/bin/wg-add-peer
chmod +x /usr/local/bin/wg-add-peer
```

## Шаг 3: Запуск скрипта

```bash
sudo ./wg-add-peer.sh
```

## Шаг 4: Следование инструкциям в TUI

1. Введите имя пира (например: `laptop-user`)
2. Выберите DNS серверы
3. Выберите режим маршрутизации
4. Подтвердите создание

## Результат

После успешного создания пира:

1. **Конфигурация выведена в STDOUT** - можно скопировать из терминала
2. **Файл создан:** `/etc/wireguard/peers_conf/<имя_пира>.conf`
3. **Пир добавлен в сервер:** `/etc/wireguard/wg0.conf` обновлен
4. **Изменения применены:** WireGuard интерфейс обновлен

## Использование конфигурации на клиенте

Скопируйте содержимое файла `/etc/wireguard/peers_conf/<имя_пира>.conf` на клиентское устройство и импортируйте в WireGuard клиент.
